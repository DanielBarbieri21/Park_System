Política de Privacidade

ParkSystem - Sistema de Estacionamento | Última atualização: 27 de junho de 2026

1. Visão geral

O ParkSystem - Sistema de Estacionamento é um aplicativo desktop para Windows destinado ao controle operacional de estacionamentos. Esta política descreve como os dados são tratados quando você utiliza o aplicativo.

2. Dados coletados

O aplicativo armazena localmente, no computador onde é instalado:

  • Placas de veículos, horários de entrada/saída e valores cobrados;
  • Credenciais de usuários (senhas protegidas por hash PBKDF2);
  • Logs operacionais em arquivo local (logs/app-YYYYMMDD.log).

O aplicativo não envia dados para servidores externos, não utiliza analytics de terceiros e não coleta informações pessoais além das necessárias para operação local.

3. Uso dos dados

Os dados são utilizados exclusivamente para:

  • Registrar entradas e saídas de veículos;
  • Calcular valores e gerar relatórios (PDF/CSV);
  • Autenticar usuários autorizados;
  • Registrar eventos técnicos para diagnóstico local.

4. Compartilhamento

Não compartilhamos, vendemos nem transferimos seus dados a terceiros. Exportações de relatório (PDF/CSV) são iniciadas pelo usuário e salvas no dispositivo local.

5. Segurança

Senhas são armazenadas com hash e salt. Recomendamos manter o sistema operacional atualizado, restringir acesso físico ao computador e alterar a senha padrão no primeiro acesso.

6. Retenção e exclusão

Os dados permanecem no banco SQLite local enquanto o aplicativo estiver instalado. Você pode excluir registros pela interface do sistema ou remover completamente os dados desinstalando o aplicativo e apagando a pasta de dados local, se desejar.

7. Crianças

Este aplicativo não é direcionado a menores de 13 anos.

8. Alterações

Esta política pode ser atualizada periodicamente. A data da última revisão será indicada no topo desta página.

9. Contato

Para dúvidas sobre privacidade, entre em contato pelo e-mail informado na listagem do aplicativo na Microsoft Store.
